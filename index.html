  • Para visualizar a base de dados de inspiração do JIRA: data.json
  • Código de seleção SQL que renderiza a prévia (o agrupamento é por front-end ou por query na API `?pretty=table/board/calendar) 
                    
                    SELECT
                        p.id AS page_id,
                        p.title AS page_title,
                        json_group_object(
                            pc.id,
                            json_object(
                            'row_id', pcv.id, 
                            'row_data', pcv.data,
                            'column_name', pc.name,
                            'column_type', pc.type,
                            'column_data', pc.data
                            )
                        ) AS page_columns
                    FROM
                        pages p
                        INNER JOIN page_hubs ph ON ph.page_id = p.id
                        LEFT JOIN page_columns_values pcv ON p.id = pcv.page_id
                        LEFT JOIN page_columns pc ON pcv.page_column_id = pc.id
                    WHERE
                        ph.page_root_id = '01KPR8D7QQCDCATHD10E449RX1'
                    GROUP BY
                        p.id,
                        p.title;
